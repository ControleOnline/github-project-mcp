# Security

Automacoes do agente de seguranca.

## Responsabilidades

- revisar tarefas associadas ao agent `Security`;
- reconhecer decisoes humanas ou do Copilot na issue e nos PRs vinculados;
- acionar o Copilot quando ainda nao houver decisao suficiente;
- repassar a task para `Quality Assurance` ou `Developer` quando a decisao estiver clara.

## Regra operacional

Quando o Copilot for acionado, a instrucao deve exigir a publicacao da decisao em formato estruturado para que o runner consiga consumir o resultado automaticamente.
